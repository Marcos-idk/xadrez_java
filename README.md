# ♟️ Jogo de Xadrez no Console (Java)

Um jogo de xadrez funcional desenvolvido em Java, executado no console, com implementação das principais regras do jogo.

---

## Funcionalidades

* Movimentação completa das peças
* Sistema de turnos (Branco vs Preto)
* Entrada no formato padrão de xadrez (ex: A2 → A4)
* Validação de movimentos
* Sistema de captura
* Detecção de xeque
* Detecção de xeque-mate
* Bloqueio de jogadas ilegais (rei não pode ficar em xeque)

---

## Conceitos Utilizados

* Programação Orientada a Objetos (POO)
* Herança e Polimorfismo
* Encapsulamento
* Simulação de regras de jogo
* Organização de código

---
